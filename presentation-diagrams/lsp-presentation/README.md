# LSP Presentation

A professional reveal.js presentation for explaining LSP Server Architecture and the TypeScript-Go implementation.

## Getting Started

1. Navigate to the presentation directory:
```bash
cd lsp-presentation
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to `http://localhost:3000`

## Features

- **Modern reveal.js**: Latest version with all plugins
- **Syntax Highlighting**: Highlight.js with Go/TypeScript support  
- **Custom Styling**: Professional theme with LSP-appropriate colors
- **Fragment Animations**: Smooth transitions and reveals
- **Speaker Notes**: Press 'S' to open speaker view
- **Navigation**: Arrow keys, space bar, or on-screen controls

## Customization

- **Themes**: Edit `custom.css` for styling
- **Content**: Add slides to `index.html`
- **Colors**: Modify CSS variables in `:root`
